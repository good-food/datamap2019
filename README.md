**备注：**
在FAO和中国统计局双方都具备数据的年份里进行比较，FAO的数据有更具体的分类，更详细的解释，最重要的是有长期的记录，且与中国统计局给出的统计结果相近（通常情况下，中国统计局的数据比FAO的略高10%，但这种差异无伤大雅），因此，我们采用FAO的数据作为本项目的数据来源。
